# Loss function
**Overview**
Description:: Measures how inaccurate the model is.

## Note

The loss function measures how inaccurate the model is. For example, the mean square error (MSE):
$$
L(\theta_0, \theta_1) = \frac{1}{m}\sum_{i=1}^{m}{(f_\theta(x_i) - y_i)^2}
$$

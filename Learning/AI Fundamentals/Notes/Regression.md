# Regression
**Overview**
Description:: An algorithm that creates a model to predict a number.

## Note

Regression is an **algorithm** that creates a model to predict a number. It is **not** the model itself.

### Why use regression?

- Works when little data is available
- Easy to interpret
- Robust
- Fast

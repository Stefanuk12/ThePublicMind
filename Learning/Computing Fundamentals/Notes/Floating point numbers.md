# Floating point numbers
**Overview**
Description:: 

## Note

Floating point numbers allow computers to store fractions and much larger (or smaller) numbers.

### Single precision

- 23 bit significand
- 8 bit exponent
- 127 offset

### Double precision

- 52 bit significand
- 11 bit exponent
- 1023 offset

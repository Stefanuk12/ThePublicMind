# Relation
**Overview**
Description:: A set of ordered pairs.

## Note

A relation is a [[Set|set]] of [[Ordered Pair|ordered pairs]].

It can be denoted like xRy or $R(x, y)$ which both mean that $x$ is related to $y$, using relation $R$.
